# 오늘경제TV — AI 자동화 스튜디오

한국 경제·주식 전문 유튜브 채널 **오늘경제TV** 제작을 위한 완전 자동화 도구 모음입니다.

---

## 📦 구성

```
오늘경제TV_스튜디오/
├── 황작가_스튜디오.py          # 메인 스튜디오 (TTS + 이미지 + 영상)
├── 오늘경제TV_대본생성기.html   # AI 대본 자동 생성기 (브라우저 실행)
├── requirements.txt
└── README.md
```

---

## 🔧 워크플로우

```
① 오늘경제TV_대본생성기.html
   뉴스/유튜브 URL 입력
   → Gemini AI 시황 분석
   → 메인 테마주 5중 검증
   → 소부장 종목 5중 검증
   → 대본 자동 생성 (5500~7700자)

② 황작가_스튜디오.py
   생성된 대본 붙여넣기
   → ElevenLabs TTS 음성 생성
   → Gemini 이미지 생성 (13화풍)
   → 음성 + 이미지 동시 생성
```

---

## 🚀 설치 및 실행

### 1. 패키지 설치

```bash
pip install -r requirements.txt
```

### 2. API 키 준비

| 도구 | API 키 발급 |
|------|------------|
| Gemini (대본 생성 + 이미지) | [aistudio.google.com/apikey](https://aistudio.google.com/apikey) |
| ElevenLabs (TTS 음성) | [elevenlabs.io](https://elevenlabs.io) |

### 3. 대본 생성기 실행

```
오늘경제TV_대본생성기.html 파일을 크롬/엣지에서 열기
```
- Gemini API 키 입력
- 뉴스 또는 유튜브 URL 추가 (최대 5개)
- 메인 테마 / 소부장 힌트 입력 (비우면 AI 자동 판단)
- **분석 + 대본 생성 시작** 클릭

### 4. 스튜디오 실행

```bash
python 황작가_스튜디오.py
```
- 브라우저에서 `http://127.0.0.1:7861` 자동 오픈
- 설정 탭에서 API 키 입력
- 생성된 대본을 TTS / 이미지 / 콤보 탭에 붙여넣기

## 🎯 대본 생성기 상세

### 5단계 파이프라인

| 단계 | 내용 |
|------|------|
| 1️⃣ URL 분석 | 뉴스/유튜브 URL에서 키워드·시사점 추출 |
| 2️⃣ 시황 재창조 | 단순 요약이 아닌 맥락·배경 붙여 재구성 |
| 3️⃣ 메인 테마주 5중 검증 | 테마연관성 / 실적모멘텀 / 수급 / 업사이드 / 리스크 |
| 4️⃣ 소부장 5중 검증 | 공급망연관성 / 수주실적 / 기술경쟁력 / 업사이드 / 리스크 |
| 5️⃣ 대본 스트리밍 생성 | 5500~7700자 / 24~30문단 / `---<` 구분자 |

### 대본 스타일 특징
- `"~거든요" "~잖아요" "~인 거예요"` 구어체
- 어려운 개념을 일상 비유로 풀어서 설명
- 실제 종목명 + 6자리 종목코드 명시
- 시황 → 종목 추천 → 아웃트로 자연스러운 흐름
